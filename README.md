# 別再只會談情說愛，來點冒險刺激吧！ 
## ---臺灣製片的商業挑戰與改善方針

>完成於2022.09.06<br>
>花費時間：一個工作天
<br>

## 一、 臺灣製片挑戰
Netflix平台上，臺灣的好評佳作超過20%，然而熱門作品卻不到5%。可見製作已經有一定品質，但觀眾卻不夠多。<br>
如何在維持影片水準下，吸引更多觀眾，打開能見度，是臺灣製片面臨的挑戰。
![image](https://github.com/Remi-KC/Solving_a_Business_Challenge/blob/main/Plot/good_pop.png)
<br>

## 二、	解決方案
### 1.	將影片題材從愛情片轉向動作冒險、科幻及懸疑片
* 撇除經典類影片本來就屬於賣座影片之後，電影作品中，熱門比例最高的是動畫、劇情、動作冒險片和科幻片。
* 電視作品中，熱門比例最高的則是懸疑、動作冒險、動畫和科幻片。
<img src="https://github.com/Remi-KC/Solving_a_Business_Challenge/blob/main/Plot/popular_genre.png" width="850"><br>
<br>

* 喜劇和愛情劇則為倒數第四名與倒數第三名，然而這卻是臺灣大部分的作品類型，難怪臺灣作品會叫好不叫座了。
<img src="https://github.com/Remi-KC/Solving_a_Business_Challenge/blob/main/Plot/TW_genre.png" width="850"><br>
<br>

* 臺灣唯一的兩部熱門作品屬於電影中熱門的類別：動作冒險和劇情類。
<img src="https://github.com/Remi-KC/Solving_a_Business_Challenge/blob/main/Plot/TWpop_table.png" width="850"><br>
* 可見影片類型的確會影響票房。未來應嘗試朝動作、科幻及懸疑片拓展。
<br>

### 2.	改為在平均上架數量較少的月份集中上架作品
* 臺灣目前每月平均上架的影片數量很少（不到5部），算是沒有特別偏好的上架月份。
* 然而Netflix本身存在上架量較多與較少的月份規律。
* 未來臺灣片商可以考慮集中在二月上架。在同期競爭者較少的情況下，不失為提高能見度的一種方法。
